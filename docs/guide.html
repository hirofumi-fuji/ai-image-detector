利用ガイド＆運用ルール

最終更新日: 2026年3月2日　｜　管理部門配布用

1. ツールの概要

本ツールは、業務で使用する画像（主にAI生成画像）の著作権リスクを自動判定するためのWebアプリケーションです。

以下の3つの分析を組み合わせて、総合的にリスクを判定します。

分析手法内容
Google Lens検索インターネット上の類似画像を自動検索し、既存の作品と一致するものがないか調べます。
AI画風分析（Gemini）AIが画像の画風を分析し、有名アーティストや既存キャラクターとの類似性を判定します。
pHash類似度画像のハッシュ値を数値的に比較し、既存画像との一致度をパーセンテージで表示します。

2. 判定結果の見方

判定意味画面表示
SAFE既存作品との類似性が低く、問題なさそうです。緑色で表示されます。
CAUTION似ている部分が見つかりました。念のため確認が必要です。黄色で表示されます。
DANGER既存作品と明らかに似ており、使用を避けるべきです。赤色で表示されます。

※ 注意: AI生成画像は多くの場合 CAUTION 判定となります。これはAIが学習データの影響を受けるためであり、必ずしも著作権侵害を意味するものではありません。

3. 使い方

  1. ツールのURL（下記）にアクセスします。
  2. 「判定する画像をアップロード」エリアに画像をドラッグ＆ドロップ、またはクリックして選択します。（複数画像の同時アップロード可）
  3. 右側の「分析開始」ボタンをクリックします。
  4. 分析が完了すると、各画像の判定結果が表示されます。
  5. 必要に応じて「レポートCSVダウンロード」で記録を保存します。

4. 判定別の対応フロー

● SAFE（緑）の場合

対応そのまま使用OK
承認不要
記録CSVレポートをダウンロードし、案件フォルダに保存

● CAUTION（黄）の場合

用途によって対応が異なります。

用途対応承認
社内利用
（企画書、ラフ案、社内プレゼン、提案資料の素案）		 下記セルフチェックリストを確認のうえ、使用OK 不要
対外利用
（クライアント納品物、広告出稿、SNS投稿、Webサイト掲載、印刷物）		 セルフチェック＋上長確認 必要（上長またはリーガル担当）

● DANGER（赤）の場合

対応使用禁止 — いかなる用途でも使用しないでください。
代替別の画像を生成し直す、またはストックフォトを使用してください。

5. CAUTION時のセルフチェックリスト

CAUTION判定の画像を使用する際は、以下の項目をすべて確認してください。

No.確認項目チェック
1「似ているアーティスト」に表示された作家の作品と見比べ、明らかな模倣ではないことを確認した□
2「ネット上の類似画像」のリンク先を確認し、既存の著作物のコピーではないことを確認した□
3画像の一致度が85%を超えている場合、該当する類似画像と目視で比較した□
4特定のキャラクター、ロゴ、商標に酷似した要素が含まれていないことを確認した□
5クライアントのブランドガイドラインに抵触しないことを確認した□

6. 記録の保管

判定結果は必ずCSVレポートとしてダウンロードし、以下の場所に保管してください。

  • 保管場所: 案件フォルダ内の「著作権チェック」フォルダ
  • ファイル名: copyright_check_report_YYYYMMDD.csv
  • 保管期限: 案件終了後2年間

万が一のクレーム対応時に「事前にチェックを実施した」ことの証跡となります。

7. 対応フローチャート

画像をアップロード → ツールで判定

  • SAFE → CSVを保存 → 使用OK
  • CAUTION → セルフチェック実施
    • 社内利用 → チェック通過で使用OK
    • 対外利用 → 上長承認 → 使用OK / 差し替え
  • DANGER → 使用禁止 → 別画像で再生成

8. よくある質問（FAQ）

質問回答
AI生成画像はほとんどCAUTIONになるのですが？正常な動作です。AIは既存の作品データを学習しているため、何らかの類似性が検出されることが多いです。セルフチェックリストを確認のうえ、問題なければ使用できます。
SAFEなら100%安全ですか？いいえ。本ツールは参考情報の提示であり、法的な安全性を保証するものではありません。ただし、リスクが低いことの一つの目安にはなります。
ストックフォト（Shutterstock等）もチェックすべきですか？正規ライセンスで購入したストックフォトは原則チェック不要です。ただし、AI生成のストックフォトを使用する場合はチェックを推奨します。
分析結果が出るまでどのくらいかかりますか？1画像あたり約10〜20秒です。複数画像は順次処理されます。
対応ブラウザは？Chrome、Edge、Safariの最新版を推奨します。
